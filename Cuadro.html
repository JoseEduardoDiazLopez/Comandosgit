CUADRO COMPARATIVO





COMANDOS
Comando Descripción Ejemplo
git init Crea un repositorio local git git init
git clone Hace una copia de un repositorio git clone usuario@host:/path/to/repository
git add Agrega un archivo a el area de preparación git add ejemplo.txt
git commit Crea una captura de los cambios y se guarda en el directorio git git commit –m “mensaje ”
git config Se usa para personalizar el usuario com oejemplo un email git config --global user.email email@ejemplo.com
git status Muestra la lista de los archivos que cambiaron junto a los que están por ser confirmados. git status
git push Se usa para enviar confirmaciones locales a la rama ma del repositorio remoto git push origin master
git checkout Crea ramas y te ayuda a navegar entre ellas command git checkout -b branch-name
git remote Te permite ver todos los repositorios remoto git remote -v
git branch Se usa para listar, crear o borrar ramas. git branch
git pull junta todos los cambios que fueron realziadoso en el repositorio remoto con el directorio local git pull
git merge Se usa para fusionar una rama con otra rama git merge branch-name
git diff Se usa para hacer una lista de conflictos git diff --base file-name
git tag Marca commits específicos git tag 1.1.0 instert-commitID-here
git log Se usa para ver el historial del repositorio listando ciertos detalles de la confirmación. git log
git reset Reinicia el index y el directorio de trabajo al último estado de confirmación git reset - -hard HEAD
git rm Remueve archivos del index y del directorio de trabajo. git rm filename.txt
git stash Guardará momentáneamente los cambios que no están listos para ser confirmados. git stash
git show Se usa para mostrar información sobre cualquier objeto git. git show
